  • Wat is machine learning:

      • Machine learning is het leren aan computers “zelfstandig” dingen te kunnen doen. Dit wordt vaak gedaan door een computer allerlei informatie te geven over een bepaald onderwerp, en het zo dingen te leren herkennen.Machine learning gaat over de ontwikkeling van software die zijn eigen performance verbetert. Machine learning leunt sterk op statistische wetenschap.

  • Methodes van machine learning:

      • Supervised learning (gecontroleerd leren): een machine learning methode waarin je werkt met gelabelde data (dataset) om voorspellingen te doen.
      Unsupervised learning (ongecontroleerd leren): geen (gelabelde) data tot je beschikking hebben om voorspellingen te kunnen meten op betrouwbaarheid.
      Semi-supervised learning (semi-gecontroleerd leren): gebruikt zowel de supervised als de unsupervised methode om eindresultaat te vormen.
      Reinforcement learning (ondersteund leren): leren van je fouten, ofwel de computer leert van z'n eigen fouten en ook andersom als het een bepaalde handeling goed uitvoert, dan zal het apparaat dat ook vaker zo op dezelfde manier toepassen.
      Transduction learning (transductie leren): wordt het minst gebruikt van alle machine learning methodes. Houdt in dat het (apparaat) zelfstandig keuzes moet leren maken tussen gevallen/problemen.
