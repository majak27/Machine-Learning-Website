Wat is machine learning?

Machine learning is het leren aan computers “zelfstandig” dingen te kunnen doen. Dit wordt vaak gedaan door een computer allerlei informatie te geven over een bepaald onderwerp, en het zo dingen te leren herkennen.Machine learning gaat over de ontwikkeling van software die zijn eigen performance verbetert. Machine learning leunt sterk op statistische wetenschap.

Methodes van machine learning?

Supervised learning (gecontroleerd leren): een machine learning methode waarin je werkt met gelabelde data (dataset) om voorspellingen te doen.

Unsupervised learning (ongecontroleerd leren): geen (gelabelde) data tot je beschikking hebben om voorspellingen te kunnen meten op betrouwbaarheid.

Semi-supervised learning (semi-gecontroleerd leren): gebruikt zowel de supervised als de unsupervised methode om eindresultaat te vormen.

Reinforcement learning (ondersteund leren): leren van je fouten, ofwel de computer leert van z'n eigen fouten en ook andersom als het een bepaalde handeling goed uitvoert, dan zal het apparaat dat ook vaker zo op dezelfde manier toepassen.

Transduction learning (transductie leren): wordt het minst gebruikt van alle machine learning methodes. Houdt in dat het (apparaat) zelfstandig keuzes moet leren maken tussen gevallen/problemen.

Hoe wordt machine learning toegepast?

  1. Gezichtsherkenning:

    Feature transformatie en feature extractie

    Bij deze stap worden bepaalde filters en pre-processing stappen uitgevoerd. De foto kan worden gesimplificeerd naar een foto met een lagere dimensionale structuur. Ook kunnen er bijvoorbeeld door middel van feature extractie bepaalde elementen uit worden voorgeselecteerd.

    Selectie van het algoritme en hyper tuning

    Na de feature transformatie en extractie wordt het algoritme geselecteerd. Dit heeft tal van verschillende subtypes, wat er voor zorgt dat er een oneindig aantal mogelijkheden. De software zoekt zo de zelf de optimale opties uit.

  2. Google Search

    3. De Google Search werkt met verschillende algoritmes. Een van deze algoritmes is het PageRank algoritme. Dit algoritme zet de resultaten op volgorde van relevantie. Dit doet het door te filteren op basis van het aantal keer dat op de website wordt geklikt.

  3. Aanbevelingssystemen

    4. Aanbevelingsalgoritmen doorlopen vaak drie verschillende fases. Als eerst verzamelt het feedback. Deze feedback kan impliciet of expliciet zijn. Het verzamelen van expliciete feedback gebeurt wanneer klanten een reactie tonen op de aanbevelingen. Impliciete feedback gaat meer over het analyseren hoe de klant de website gebruikt; hoe vaak je bijvoorbeeld naar een bepaald genre van films kijkt. Vervolgens leert het algoritme van deze verzamelde informatie; Netflix weet nu bijvoorbeeld dat jij misschien ontzettend veel van romcoms kunt genieten. Ten slotte voorspelt het algoritme op basis van deze kennis een aanbeveling die wellicht in de smaak zou kunnen vallen bij jou.
