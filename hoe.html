Hoe wordt machine learning toegepast?

Machine learning wordt tegenwoordig op veel verschillende vlakken toegepast. Machine learning is namelijk heel handig voor allerlei verschillende soorten stukken technologie. Doordat de software nu kan leren van zijn eigen fouten, kunnen er veel nuttige toevoegingen worden gedaan aan deze software. Hieronder kun je over een paar voorbeelden van machine learning lezen die je in het dagelijks leven vaak terug ziet komen.

  1. Gezichtsherkenning

    2. Veel nieuwere telefoons hebben de functie gezichtsherkenning. Hierdoor kun je bijvoorbeeld je telefoon ontgrendelen door alleen maar je gezicht te laten scannen. Dit is best handig voor het snel en veilig beschermen van je telefoon. Maar hoe werkt dit nou precies? Een aantal stappen gaan hieraan vooraf.

    Stap 1: Feature transformatie en feature extractie
    Je telefoon kan natuurlijk niet zomaar je gezicht herkennen. Je gezicht is een erg ingewikkelde vorm, en om er iets mee te kunnen doen moet het dus simpeler worden gemaakt. Bij deze eerste stap worden er dus ook bepaalde filters en pre-processing stappen uitgevoerd. De foto kan worden gesimplificeerd naar een foto met een lagere dimensionale structuur. Ook kunnen er bijvoorbeeld door middel van feature extractie bepaalde elementen uit worden voorgeselecteerd. Zo scant de software het gezicht en slaat de belangrijkste dingen op om er vervolgens iets van te maken wat het kan begrjipen.

    Stap 2: Selectie van het algoritme en hyper tuning
    Het gezicht is nu dus gescant, maar nu moet er nog iets mee gedaan worden. Is het wel het gezicht van de eigenaar van de telefoon? Na de feature transformatie en extractie wordt er een algoritme geselecteerd om dit te besluiten. Dit algoritme heeft veel verschillende subtypes, wat voor een oneindig aantal mogelijkheden zorgt. De software zoekt zo de zelf de beste optie uit. Dit doet het op basis van eerdere ervaringen; Machine learning dus!

  2. Google Search

    3. De meest gebruikte vorm van machine learning is waarschijnlijk de Google search optie. Elke seconde worden er zo'n 40,000 zoekopdrachten gedaan. Dat zijn er 3,5 miljard per dag en 1,2 biljoen per jaar. De reden dat deze zoekmachine zo goed werkt is omdat het gebruikt maakt van, je raadt het al, machine learning. De Google Search optie werkt met verschillende algoritmes. Een van deze algoritmen is het PageRank algoritme. Dit algoritme zet de resultaten op volgorde van relevantie. Dit doet het door te filteren op basis van het aantal keer dat op de website wordt geklikt.

  3. Aanbevelingssystemen

    4. Een reden waarom vermaak applicaties vaak zo verslavend zijn is vanwege de onderliggende algoritmen die precies aanbevelen waar jij naar op zoek bent. Hoe gebeurt dit? Aanbevelingsalgoritmen doorlopen vaak drie verschillende fases. Allereerst verzamelt het feedback. Deze feedback kan impliciet of expliciet zijn. Het verzamelen van expliciete feedback gebeurt wanneer klanten een reactie tonen op de aanbevelingen. Impliciete feedback gaat meer over het analyseren hoe de klant de website gebruikt; hoe vaak je bijvoorbeeld naar een bepaald genre van films kijkt. Vervolgens leert het algoritme van deze verzamelde informatie; Netflix weet nu bijvoorbeeld dat jij misschien ontzettend veel van romcoms kunt genieten. Ten slotte voorspelt het algoritme op basis van deze kennis een aanbeveling die wellicht in de smaak zou kunnen vallen bij jou.
