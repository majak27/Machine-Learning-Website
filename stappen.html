De stappen van machine learning:

Het Machine Learning proces is eigenlijk het bouwen van een model dat kan worden gebruikt om een ​​oplossing voor een probleem te vinden. Laten we aannemen dat je een probleem hebt dat moet worden opgelost met behulp van Machine Learning...

De onderstaande stappen worden gevolgd in een Machine Learning proces:

  1. het verzamelen van gegevens: Machine learning heeft historische data nodig (data uit het verleden). Je kan hierbij jezelf vragen afvragen zoals: Wat voor soort data is nodig om dit probleem op te lossen? Hoe kan ik ervoor zorgen dat die data wordt onthouden? etc.

  2. De tweede stap is de bewerken van de data, omdat veel van de data niet direct kan worden gebruikt. Maar het bewerken van essentiële delen van die data kan leiden tot onjuiste berekeningen of onjuiste voorspellingen.

  3. De volgende stap is om een ​​model te kiezen. Dus wat voor soort algoritme en wat voor model binnen het algoritme moet je gebruiken? Het kiezen van het juiste bepaalde soort algoritme hangt af van het type probleem dat je probeert op te lossen.

  4. Vervolgens moeten we het model trainen om een getraind-model te creeëren die steeds sneller fouten kan ontdekken en ook kan herstellen. Dan moeten we het model ook nog testen om er zeker van te zijn dat het correct berekent en/of voorspelt.

  5. Tenslotte na het trainen moet het model nog worden getest. De test-dataset wordt gebruikt om de efficiëntie van het model te controleren en hoe nauwkeurig het de uitkomst kan voorspellen. Nadat de nauwkeurigheid is berekend, kan je nog kijken naar enige verdere nodige verbeterpunten van het model.
